# 商业原理库

## 经济学原理

### 1. 稀缺原理
- **定义**：资源有限而欲望无限，产生选择问题
- **公式**：`价值 = 需求 / 稀缺性`
- **泛化应用**：定价策略、时间管理、注意力经济、职业选择

### 2. 供需原理
- **定义**：价格由供给和需求的交点决定
- **公式**：`P* = 供给曲线与需求曲线交点`
- **泛化应用**：市场分析、招聘市场、约会市场、内容生产

### 3. 边际原理
- **定义**：决策基于增量收益与增量成本的比较
- **公式**：`最优：边际收益 = 边际成本`
- **泛化应用**：产品功能优先级、任务排序、资源分配

### 4. 机会成本原理
- **定义**：选择某事物的代价是放弃的最佳替代选项
- **公式**：`真实成本 = 显性成本 + 隐性机会成本`
- **泛化应用**：职业选择、投资决策、时间分配

## 心理学原理

### 1. 动机原理
- **定义**：行为由内在驱动（马斯洛需求层次）
- **公式**：`动机强度 = 需求紧迫性 × 目标可达性`
- **泛化应用**：产品设计、用户激励、员工管理、自我成长

### 2. 认知失调原理
- **定义**：态度与行为不一致时产生心理不适
- **公式**：`失调 = |态度 - 行为|`
- **泛化应用**：销售技巧、行为改变、用户转化

### 3. 社会证明原理
- **定义**：人们通过他人行为判断正确性
- **公式**：`信任 = 他人的数量 × 他人的权威性`
- **泛化应用**：用户评价、推荐系统、品牌建设

### 4. 锚定效应
- **定义**：第一印象影响后续判断
- **公式**：`判断 = 初始锚点 + 调整`
- **泛化应用**：定价策略、谈判、谈判、产品设计

## 管理学原理

### 1. 分工原理
- **定义**：专业化分工提高效率
- **公式**：`总效率 = Σ(专业效率 × 熟练度)`
- **泛化应用**：团队协作、组织设计、流程优化

### 2. 委托-代理原理
- **定义**：代理人行为可能与委托人利益不一致
- **公式**：`代理成本 = 信息不对称 × 目标差异`
- **泛化应用**：公司治理、激励机制、外包管理

### 3. 规模原理
- **定义**：规模增加先带来效率后带来不经济
- **公式**：`单位成本 = f(规模)`（先降后升）
- **泛化应用**：扩张策略、组织设计、产品规划

### 4. 路径依赖原理
- **定义**：过去选择限制未来可能性
- **公式**：`当前选择 = 历史路径 + 约束`
- **泛化应用**：战略转型、技术选择、职业发展

## 市场营销原理

### 1. 价值主张原理
- **定义**：产品为用户提供的独特价值
- **公式**：`价值 = (功能 + 情感) / 成本`
- **泛化应用**：产品定位、个人品牌、求职

### 2. 用户旅程原理
- **定义**：用户从认知到购买的完整路径
- **公式**：`转化率 = Π(各步骤转化率)`
- **泛化应用**：转化优化、内容营销、销售流程

### 3. 网络效应原理
- **定义**：用户增加提升产品价值
- **公式**：`价值 ∝ 用户数 × 连接度`
- **泛化应用**：平台策略、社交产品、社区运营

### 4. 品牌原理
- **定义**：品牌是用户心智中的定位和联想
- **公式**：`品牌资产 = 知名度 × 美誉度 × 差异化`
- **泛化应用**：个人品牌、产品品牌、企业品牌

## 创新原理

### 1. 第一性原理创新
- **定义**：从最基本真理出发重新思考
- **公式**：`创新 = 去除假设 + 回归本质 + 重构方案`
- **泛化应用**：产品创新、商业模式创新、问题求解

### 2. 迭代原理
- **定义**：通过快速试错和反馈改进
- **公式**：`V_n+1 = V_n + ΔV(反馈)`
- **泛化应用**：敏捷开发、产品迭代、个人成长

### 3. 组合创新
- **定义**：将现有元素以新方式组合
- **公式**：`创新 = 元素A + 元素B + 新连接`
- **泛化应用**：跨界融合、创意生成、解决方案设计

### 4. 逆向创新
- **定义**：从限制出发寻找突破
- **公式**：`创新 = 从不可能反推可行`
- **泛化应用**：低成本创新、资源受限创新、颠覆式创新
