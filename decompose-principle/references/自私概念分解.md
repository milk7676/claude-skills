```
极端利他 ← 合理自爱 → 自私 → 恶性自私

向量表示：
行为倾向 = α·利他 + β·自爱 + γ·自私 + δ·恶性自私

健康状态：
  α + β >> γ + δ
  (利他 + 自爱 >> 自私 + 恶性)

病理状态：
  γ >> β 或 δ >> α
  (自私 >> 自爱 或 恶性 >> 利他)
```

### 自私的生物学公式

```
基因视角：
  基因存活 = 个体存活 + 后代存活 + 亲属存活

亲缘选择（汉密尔顿法则）：
  利他行为条件：rB > C
  其中：
  r = 亲缘关系系数（0.5子女，0.25孙辈）
  B = 受益者的适应度提升
  C = 施予者的适应度损失

  结论：对亲属利他 = 基因层面的自私
```

### 自私 vs 自爱

```
自爱 =
  满足基本需求 +
  保护边界 +
  发展自我 +
  尊重他人

自私 =
  优先自己 +
  忽视他人 +
  侵犯边界 +
  零和思维

向量差异：
  自爱 = (+自己, +他人)
  自私 = (+自己, -他人)
```

### 类比推理

## 八、实践应用

### 自我觉察：检测自己的自私程度

```

1. 做决定时，我考虑他人的影响吗？

  多数"是" → 自私程度较低
  多数"否" → 需要反思
```

### 平衡自私与利他

```
原则：
  先自爱，再利他

具体：
  1. 满足基本需求（不是自私，是必要）
  2. 发展自身能力（更好的自己才能更好地帮助他人）
  3. 有余力时帮助他人（可持续的利他）
  4. 设置边界（保护自己，也保护关系）

向量公式：
  健康行为 = (自爱 × 0.5) + (利他 × 0.5)
```

### 培养健康自爱

```
区分：
  自私 = 我要赢，你输
  自爱 = 我要赢，你也可以赢

行动：
  · 确认自己的需求（不是自私）
  · 表达自己的边界（不是自私）
  · 追求自己的发展（不是自私）
  · 同时尊重他人的需求
```
