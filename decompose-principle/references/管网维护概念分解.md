## 二、向量公式

### 核心表示

```
管网维护 = f(资产, 状态, 风险, 资源, 策略)

其中：
资产 = Σ(管段 × 权重1)

```

### 决策优化

```
最优维护策略 = max(维护效果 / 维护成本)

维护效果 =
  α·可靠性提升 +
  β·漏损减少 +
  γ·水质改善 +
  δ·寿命延长

成本 =
  人力成本 +
  材料成本 +
  机会成本 +
  风险成本
```

### 风险评估

```
管段风险值 =
  (管龄因子 × 年龄权重) +
  (材质因子 × 材质权重) +
  (环境因子 × 环境权重) +
  (负荷因子 × 负荷权重)

维护优先级 =
  风险值 × 重要度权重 × 紧迫度权重
```

## 六、关键启发式

### 压缩启发式

```
二八原则：
  20%的管段承载了80%的维护需求
  → 识别关键资产，集中资源

浴盆曲线：
  早期故障期 → 稳定期 → 衰退期
  → 不同阶段采用不同策略

DMA分区：
  整体管网 → 分区管理
  → 降低复杂度，提高可管理性
```

### 风险启发式

```
风险矩阵：
  高概率 × 高影响 = 优先维护
  高概率 × 低影响 = 计划维护
  低概率 × 高影响 = 重点监测
  低概率 × 低影响 = 延后维护

重要度因子：
  医院、学校等关键用户 = 权重加倍
  干管、主干道 = 权重加倍
```

### 预测启发式

```
基向量选择：
  预测模型 = f(管龄, 材质, 环境, 历史, 实时)

  高置信度预测 → 自动触发维护
```

## 七、实用框架

### 风险优先数（RPN）方法

```
RPN = 严重度(S) × 发生频率(O) × 检测难度(D)

其中：
S (Severity) = 故障影响程度（1-10）
O (Occurrence) = 故障发生概率（1-10）
D (Detection) = 故障检测难度（1-10）

应用：
  RPN > 100：高优先级
  50 < RPN ≤ 100：中优先级
  RPN ≤ 50：低优先级
```

## 关键洞察

```
所有基础设施维护的根节点都是相同的：
  在有限资源下，最大化资产价值

只是"参数"不同：
  - 物理参数（管材 vs 线路）
  - 监测参数（压力 vs 电压）
  - 风险参数（漏水 vs 停电）

但"公式"是一样的：
  风险 = 概率 × 影响
  最优 = max(效果/成本)
  演进 = 经验 → 智能
```
