# 科学原理库

## 物理学原理

### 1. 守恒原理
- **定义**：孤立系统中某物理量总量不变
- **公式**：`能量守恒：E_before = E_after`
- **泛化应用**：质量守恒、动量守恒、信息守恒（量子力学）、注意力守恒

### 2. 相对性原理
- **定义**：物理定律在所有惯性系中相同
- **公式**：`惯性系变换保持定律不变`
- **泛化应用**：文化相对主义、价值多元、框架选择

### 3. 量子原理
- **定义**：微观世界具有概率性和叠加性
- **公式**：`Ψ = Σ(α_n × ψ_n)`（态叠加）
- **泛化应用**：不确定性管理、并行决策、期权思维

### 4. 涨落-耗散原理
- **定义**：系统偏离平衡时会产生恢复趋势
- **公式**：`偏离越大，恢复力越强`
- **泛化应用**：价格回归价值、过度矫正、均值回归

## 数学原理

### 1. 极限原理
- **定义**：通过无限逼近获得精确结果
- **公式**：`lim(x→a) f(x) = L`
- **泛化应用**：渐进学习、迭代改进、长期趋势

### 2. 最优化原理
- **定义**：在约束条件下寻找极值
- **公式**：`max/min f(x) subject to g(x) = 0`
- **泛化应用**：资源分配、决策优化、帕累托改进

### 3. 概率原理
- **定义**：用概率描述不确定性
- **公式**：`P(A|B) = P(B|A) × P(A) / P(B)`（贝叶斯）
- **泛化应用**：风险评估、信息更新、决策理论

### 4. 图论原理
- **定义**：用点和边表示关系
- **公式**：`图 G = (V, E)`
- **泛化应用**：社交网络、供应链、知识图谱、组织架构

## 生物学原理

### 1. 进化原理
- **定义**：通过变异、选择、保留适应环境
- **公式**：`新一代 = (旧一代 + 变异) × 选择`
- **泛化应用**：产品迭代、组织进化、创意筛选、技术演进

### 2. 代谢原理
- **定义**：生命通过物质能量交换维持
- **公式**：`生命 = 输入能量 - 损耗 + 转化`
- **泛化应用**：商业模式、资源循环、知识新陈代谢

### 3. 发育原理
- **定义**：基因蓝图在环境中实现
- **公式**：`表型 = 基因型 × 环境`
- **泛化应用**：个人成长、产品上市、战略实施

### 4. 共生原理
- **定义**：不同物种互利共存
- **公式**：`双方收益 > 个体独立收益`
- **泛化应用**：生态系统、商业联盟、协作网络

## 认知科学原理

### 1. 注意力原理
- **定义**：认知资源有限且可分配
- **公式**：`总注意力 = Σ(分配给任务的注意力)`
- **泛化应用**：产品设计、内容营销、用户体验

### 2. 记忆原理
- **定义**：信息通过编码、存储、检索维持
- **公式**：`记忆 = 编码 × 存储强度 × 检索线索`
- **泛化应用**：学习策略、知识管理、品牌记忆

### 3. 启发式原理
- **定义**：用简化规则快速决策
- **公式**：`决策 ≈ 启发式规则`
- **泛化应用**：产品设计、UX设计、行为经济学

### 4. 心智模型原理
- **定义**：人用内部模型解释世界
- **公式**：`解释 = 心智模型 × 观察数据`
- **泛化应用**：沟通策略、产品设计、叙事框架
