# 供水管网设备管理

## 设备类型分类

### 阀门类
- 闸阀（DN50-DN1200）
- 蝶阀（DN50-DN1400）
- 球阀（DN15-DN300）
- 排气阀、泄水阀

### 水表类
- 大口径水表（DN50-DN300）
- 小口径水表（DN15-DN40）
- 流量计

### 泵站设备
- 离心泵
- 变频器
- 控制柜
- 管路系统

### 调压设施
- 减压阀
- 稳压阀
- 调节池

## 设备台账信息

### 基本信息
| 字段 | 说明 |
|------|------|
| 设备编号 | 唯一标识（如：V-001-2024） |
| 设备名称 | 具体名称 |
| 设备类型 | 阀门/水表/水泵等 |
| 规格型号 | DN、材质、压力等级 |
| 安装位置 | 坐标、路段 |
| 安装日期 | YYYY-MM-DD |
| 制造商 | 厂家名称 |
| 出厂编号 | 厂家序列号 |
| 使用年限 | 预计使用年数 |

### 状态信息
| 字段 | 说明 |
|------|------|
| 当前状态 | 正常/维修/停用/报废 |
| 健康评分 | A-优秀/B-良好/C-一般/D-需关注 |
| 最近维护日期 | YYYY-MM-DD |
| 下次维护日期 | YYYY-MM-DD |
| 累计运行时间 | 小时数 |
| 故障次数 | 累计故障次数 |

## 维护类型

### 日常维护
- **频率**：每月
- **内容**：外观检查、清洁、润滑、紧固
- **耗时**：15-30分钟/台

### 定期维护
- **频率**：每季度/半年/一年
- **内容**：性能检测、部件更换、系统校准
- **耗时**：2-4小时/台

### 预防性维护
- **频率**：根据设备年限
- **内容**：提前更换易损件、深度检修
- **耗时**：1-2天/台

### 故障维修
- **触发**：故障发生
- **内容**：故障诊断、部件更换、功能恢复
- **耗时**：根据故障类型

## 设备生命周期管理

### 新建阶段
1. 设备选型与采购
2. 验收与测试
3. 建立台账
4. 录入系统

### 运行阶段
1. 日常巡检记录
2. 定期维护执行
3. 故障维修记录
4. 性能数据积累

### 评估阶段
1. 健康状况评估
2. 维护成本分析
3. 更新改造建议
4. 报废判定

### 报废/更换阶段
1. 报废审批
2. 拆除回收
3. 台账更新
4. 新设备采购

## 维护计划制定

### 年度维护计划
- 上一年12月制定
- 覆盖所有设备
- 平衡维护资源
- 预留应急时间

### 月度维护计划
- 每月25日前制定
- 优先安排高风险设备
- 考虑天气因素
- 准备备件清单

## 设备健康评估

### 评分维度
- 运行稳定性（40%）
- 维护频率（30%）
- 故障历史（20%）
- 使用年限（10%）

### 评分标准
| 评分 | 等级 | 说明 |
|------|------|------|
| 90-100 | A | 状态优秀，正常维护 |
| 75-89 | B | 状态良好，需关注 |
| 60-74 | C | 状态一般，需加强维护 |
| <60 | D | 状态差，建议更换 |
