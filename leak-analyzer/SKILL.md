# 供水管网漏水分析

## 分析方法

### 1. 夜间最小流量分析（MNF）
- 选取凌晨2:00-4:00时段数据（用水量最小）
- MNF值 = 该时段平均流量
- 可疑漏水判定：MNF > 基准值 × 1.5倍

### 2. DMA分区计量分析
- 分区总表与各用户表读数对比
- 计算漏损率 = (总表读数 - 各用户表读数之和) / 总表读数 × 100%
- 正常漏损率 < 15%，> 25% 需排查

### 3. 压力异常分析
- 突然压力下降：可能存在爆管
- 局部压力偏低：可能存在隐蔽泄漏
- 压力波动异常：可能存在阀门问题

### 4. 听音检测
- 利用听音杆、相关仪检测漏水声音
- 频率特征：漏水声通常在 500-2000 Hz
- 声音衰减：土壤中约 5-15米/100dB

## 需要的数据

### 必需数据
- 管网拓扑图（管径、材质、长度）
- 检测时段压力数据（时间、地点、压力值）
- 检测时段流量数据（时间、地点、流量值）

### 辅助数据
- 水表历史读数
- 用水规律（工业/居民/商业）
- 近期施工记录
- 地质情况

## 漏水定位流程

1. **数据收集**：获取目标区域压力、流量数据
2. **异常识别**：找出压力低、流量异常的点位
3. **范围缩小**：结合管网拓扑图，确定疑似漏水区域
4. **现场检测**：使用听音设备精确定位
5. **验证确认**：开挖验证或使用相关仪确认

## 常见漏水类型

| 类型 | 特征 | 检测难度 |
|------|------|---------|
| 管道爆裂 | 压力骤降、流量激增 | 容易 |
| 管道裂缝 | 压力持续偏低、流量持续偏大 | 中等 |
| 接口泄漏 | 局部压力低、听音明显 | 中等 |
| 阀门泄漏 | 阀门下游压力偏低 | 较难 |
| 用户私接 | 长期流量异常 | 困难 |
