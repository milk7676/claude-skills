# 供水管网维修工单

## 工单类型

### 抢修类工单
- **触发**：突发故障、紧急事件
- **优先级**：P0（最高）
- **响应时间**：30分钟内到达现场
- **完成时限**：24小时内
- **适用场景**：爆管、严重漏水、停水事件

### 维修类工单
- **触发**：巡检发现、日常维修
- **优先级**：P1
- **响应时间**：4小时内
- **完成时限**：48小时内
- **适用场景**：阀门维修、小口径管道修补、设施更换

### 改造类工单
- **触发**：计划改造、更新项目
- **优先级**：P2
- **响应时间**：按计划执行
- **完成时限**：按项目计划
- **适用场景**：管道更换、管网改造、设施升级

### 安装类工单
- **触发**：新装申请、扩容需求
- **优先级**：P3
- **响应时间**：5个工作日内
- **完成时限**：15个工作日内
- **适用场景**：新用户接水、水表安装、管道接入

### 巡检类工单
- **触发**：定期巡检、专项检查
- **优先级**：P4
- **响应时间**：按计划执行
- **完成时限**：按计划完成
- **适用场景**：日常巡检、设施检查、隐患排查

## 工单基本信息

### 工单编号规则
格式：WO-YYYYMMDD-TYPE-XXX

示例：WO-20240115-EMG-001

- WO：工单前缀
- YYYYMMDD：创建日期
- TYPE：类型（EMG=抢修，REP=维修，RNO=改造，INS=安装，INS=巡检）
- XXX：当天序号（001-999）

### 工单基本信息
| 字段 | 说明 | 示例 |
|------|------|------|
| 工单编号 | 唯一标识 | WO-20240115-REP-005 |

| 优先级 | P0/P1/P2/P3/P4 | P1 |

## 工单验收

### 验收标准
| 验收项 | 标准 | 结果 |
|-------|------|------|
| 故障是否消除 | 是/否 | 是 |
| 功能是否正常 | 是/否 | 是 |
| 现场是否清理 | 是/否 | 是 |
| 是否符合安全规范 | 是/否 | 是 |

### 验收记录
| 字段 | 说明 | 示例 |
|------|------|------|
| 验收时间 | YYYY-MM-DD HH:MM | 2024-01-15 16:00 |
| 验收人员 | 验收人 | 王班长 |
| 验收结果 | 合格/不合格 | 合格 |
| 验收意见 | 具体评价 | 阀门功能恢复，现场清理到位，通过验收 |
| 申报人确认 | 申报人签字 | 李四（电话确认满意） |

### 问题处理
- **验收合格**：工单完成，归档
- **验收不合格**：返工，重新提交验收
- **部分完成**：创建新工单处理遗留问题

## 工单归档

### 归档内容
- 工单基本信息
- 派发记录
- 执行记录
- 材料设备清单
- 验收记录
- 费用结算
- 现场照片/视频
- 相关文件（发票、合同等）

### 归档要求
- 工单完成后3个工作日内归档
- 电子档案和纸质档案同步
- 档案保存期限：5年
- 重大事件档案：长期保存
