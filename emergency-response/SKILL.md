# 供水管网应急响应

## 事件分级

### Ⅰ级（特别重大）
- DN300及以上主管道爆裂
- 影响人口超过1万人
- 停水时间超过24小时
- 涉及水质安全事件

### Ⅱ级（重大）
- DN200-DN300主管道故障
- 影响人口3000-1万人
- 停水时间8-24小时
- 重大设备故障

### Ⅲ级（较大）
- DN100-DN200管道故障
- 影响人口1000-3000人
- 停水时间4-8小时

### Ⅳ级（一般）
- DN100以下管道故障
- 影响人口1000人以下
- 停水时间4小时以内

## 应急响应流程

### 第一阶段：接报与研判（5分钟内）
1. 接到报警/发现异常
2. 初步判断事件等级
3. 确认影响范围
4. 启动相应等级应急响应

### 第二阶段：现场控制（30分钟内）
1. 派遣人员赶赴现场
2. 关闭上游阀门控制水源
3. 设置警戒线和警示标识
4. 疏散受影响人群

### 第三阶段：应急供水（1小时内）
1. 通知受影响用户
2. 启动应急供水方案
   - 应急送水车
   - 消防栓取水点
   - 临时供水设施
3. 重点单位优先保障（医院、学校等）

### 第四阶段：抢修处置
1. 制定抢修方案
2. 调配抢修人员与设备
3. 实施抢修作业
4. 实时通报进展

### 第五阶段：恢复供水
1. 抢修完成验收
2. 管道冲洗消毒
3. 水质检测合格
4. 分区域恢复供水
5. 通知用户

### 第六阶段：善后与总结
1. 现场清理
2. 赔偿评估与处理
3. 事件调查分析
4. 编写应急总结报告
5. 提出改进建议

## 常见应急事件处置

### 管道爆裂
1. 立即关闭上游阀门
2. 排水、开挖
3. 检查损坏情况
4. 制定修复方案（更换/修补）
5. 恢复供水前冲洗消毒

### 阀门失效
1. 上游关阀
2. 更换或修复阀门
3. 管网冲洗
4. 恢复供水

### 水质异常
1. 立即停水
2. 取样检测
3. 追查污染源
4. 管网冲洗消毒
5. 连续检测合格后恢复供水

### 泵站故障
1. 启用备用泵/备用电源
2. 检修故障设备
3. 调整供水方案
4. 确保基本供水

### 第三方施工破坏
1. 责令停止施工
2. 保护现场
3. 协商修复责任
4. 依法追偿

| 事件等级 | 人数 | 配置 |
|---------|------|------|
| Ⅰ级 | 10+ | 管道工、焊工、电工、起重工、安全员 |
| Ⅱ级 | 6-10 | 管道工、焊工、电工 |
| Ⅲ级 | 3-6 | 管道工、焊工 |
| Ⅳ级 | 2-3 | 管道工 |

### 抢修设备
- 挖掘机、装载机
- 发电机、水泵
- 电焊机、切割机
- 吊装设备
- 照明设备
- 临时供水车

### 材料储备
- 各口径管材
- 法兰、三通、弯头
- 阀门
- 密封材料
- 支撑材料

## 预案演练
