# 供水水质监测报告

## 常规检测指标

### 感官性状指标
| 指标 | 单位 | 标准 | 检测方法 |
|------|------|------|---------|
| 浑浊度 | NTU | ≤1.0 | 散射法 |
| 色度 | 度 | ≤15 | 铂钴标准比色法 |
| 臭和味 | - | 无异臭异味 | 嗅觉法 |
| 肉眼可见物 | - | 无 | 目视法 |
| pH值 | - | 6.5-8.5 | 玻璃电极法 |

### 一般化学指标
| 指标 | 单位 | 标准 | 检测方法 |
|------|------|------|---------|
| 总硬度 | mg/L | ≤450 | EDTA滴定法 |
| 铁 | mg/L | ≤0.3 | 原子吸收法 |
| 锰 | mg/L | ≤0.1 | 原子吸收法 |
| 铜 | mg/L | ≤1.0 | 原子吸收法 |
| 锌 | mg/L | ≤1.0 | 原子吸收法 |
| 硫酸盐 | mg/L | ≤250 | 重量法 |
| 氯化物 | mg/L | ≤250 | 硝酸银滴定法 |

### 毒理学指标
| 指标 | 单位 | 标准 | 检测方法 |
|------|------|------|---------|
| 氟化物 | mg/L | ≤1.0 | 离子选择电极法 |
| 硝酸盐 | mg/L | ≤10 | 酚二磺酸分光光度法 |
| 砷 | mg/L | ≤0.01 | 氢化物原子荧光法 |
| 铬(六价) | mg/L | ≤0.05 | 二苯碳酰二肼分光光度法 |

### 微生物指标
| 指标 | 单位 | 标准 | 检测方法 |
|------|------|------|---------|
| 菌落总数 | CFU/mL | ≤100 | 平板计数法 |

| 指标 | 单位 | 标准 | 检测方法 |
|------|------|------|---------|

| 三卤甲烷 | mg/L | ≤1.0 | 气相色谱法 |
| 四氯化碳 | mg/L | ≤0.002 | 气相色谱法 |

## 采样点设置

### 源水采样点
- 取水口（1个）
- 取水口上游（2个）

### 出厂水采样点
- 沉淀池出水（每个池）
- 滤池出水（每个池）
- 清水池出水（每个池）

### 管网采样点
- 管网起点（1个）
- 管网中间（2-3个）
- 管网末梢（2个）
- 二次供水设施（每个小区）

### 重点监控点
- 医院、学校等重点单位
- 老旧管网区域
- 易受污染区域
- 用户投诉集中区域

## 采样要求

### 采样频率
| 采样点类型 | 检测频率 |
|-----------|---------|
| 源水 | 每日1次（常规），每周1次（全项） |
| 出厂水 | 每日2次（常规），每月1次（全项） |
| 管网 | 每月2次（常规），每季1次（全项） |
| 二次供水 | 每半年1次 |

### 采样方法
1. 灭菌瓶采样（微生物指标）
2. 清洁瓶采样（理化指标）
3. 采样前放水3-5分钟
4. 样品4℃冷藏保存
5. 4小时内送检

## 水质异常处理

### 浑浊度超标
1. 增加混凝剂投加量
2. 检查滤池运行状态
3. 清洗滤池
4. 排泥处理

### 余氯不足
1. 增加消毒剂投加量
2. 检查消毒设备
3. 清洗清水池
4. 管网末端二次加氯

### 微生物超标
1. 立即停止供水
2. 增加消毒剂投加
3. 管网冲洗消毒
4. 连续检测合格后恢复

### 重金属超标
1. 追查污染源
2. 调整水源
3. 增加处理工艺
4. 通知用户

### 臭味异常
1. 检查水源状况
2. 增加活性炭处理
3. 调整消毒工艺
4. 排查管网污染
